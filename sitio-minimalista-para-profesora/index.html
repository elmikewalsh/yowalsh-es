Sitio minimalista para profesora

Geraldine Ovalle solicitó un nuevo diseño para su sitio web en preparación del giro y enfoque en sus nuevos proyectos. El nuevo sitio web necesitaba ser limpio, moderno, minimalista y rápido. También era necesario la optimización para búsquedas en internet, para mayor visibilidad.

Para este proyecto seleccioné como base del sitio la plantilla Splitian del sistema de gestión de contenido (CMS) Publii. Luego traje los elementos esenciales del sitio anterior al nuevo diseño. Conservamos el logo emblemático de las versiones anteriores.

Además del CV, clases y talleres presentes en la versión anterior del sitio web, agregamos una sección para sus publicaciones, las cuales incluyen el sitio web Huellas y Sabores, el libro para el proyecto Fondart y el eLibro: Huellas y Sabores.

Detalles técnicos

  • Color destacado es el morado de versiones anteriores - #7D14C0
